    7天学会ECS

    第一课链接：文档【7天学会ECS】
    前四部分为Class2-4的总结。

    开通云服务器

    购买网址：https://ecs-buy.aliyun.com/#/prepay

    1.计费方式

    包年包月：预付费产品，先购买数月/年ECS产品，后使用。适用于长期稳定使用服务器的场景。例如做网站。
    按量付费：后付费产品，保证有100元余额后按量（每小时花费）付费。适用于测试或跑数据。

    2.地域

    地域 (Region)：服务器数据中心所在的城市。例如华东1（杭州）、华东2（上海）。地域之间的ECS不可内网互联。
    可用区 (Zone)：同一地域下数据中心所在的地区。距离近，有光纤连接，内网可互联。

    购买ECS时建议选择业务中访客最多、转化率最高的地域，而非离你最近的。

    3.网络

    经典网络（早期ECS使用）：同一地域内的所有ECS、OSS、RDS可互联，并通过安全组实现访问的控制。
    专有网络（推荐）：用户自有一个虚拟路由网络与交换机，同一网络和交换机下的ECS、OSS、RDS等才可以互联。另外，可以绑定弹性IP，方便更换ECS时保持IP不变。

    4.实例系列

    实例：云服务器的规格。包括CPU、内存等，按需选择。

    5.公网带宽

    按流量计费：按具体使用流量（每GB）收费，不同地域的资费标准不同。
    按带宽计费：先买断带宽(M)，流量不计费。建议至少有2M带宽，配合OSS放置资源。

    6.镜像

    公共镜像：阿里云官方提供的公共基础镜像。仅包括初始系统环境，需自主配置应用环境和软件配置。
    自定义镜像：由用户系统快照生成。包括初始系统环境、应用环境、软件配置，可以节省配置时间。
    共享镜像：其他账号共享给你的自定义镜像。用户自行承担镜像完整性与安全性的风险。
    镜像市场：经严格审核的优质镜像。预装操作系统、应用环境、软件，无需配置（一键部署），使用于建站、应用开发、可视化管理等个性化场景。

    7.存储

    系统盘：操作系统所安装的虚拟硬盘。
    数据盘：提供挂载到某一盘符或目录下。安全可靠性更高，适用于存放网站数据。
    普通云盘（早期ECS使用）：三盘副本，高可靠性，但性能孱弱。只用于无I/O优化的实例。
    高效云盘：三盘副本，高可靠性，拥有1240-3000 IOPS的磁盘性能。60G以下的性价比高。
    SSD云盘：容量越大性能越好，最大20000 IOPS。60G以上性能价值明显。

    8.其他

    购买时长：选择1周/1-9个月/1-5年，是否自动续费。
    台数：选择购买多少台相同配置的ECS。

    9.安全设置

    设置密钥：通过密钥文件登入Linux系统。
    自定义密码（推荐新手）：通过密码登入Linux或Windows系统的root账号。
    创建后设置（不推荐）：生成实例后自行重置密码。

    1.开通ECS云服务器

    2.安装网页运行环境

    镜像：

    主流建站环境为LAMP (Linux + Apache Httpd + MySQL + PHP)环境，兼容绝大部分HTML和PHP运行需求。

    在镜像市场中选择LAMP 环境（Ubuntu16.04 Apache PHP7.1），它提供OSS备份、Let’s Encrypt等，并允许Apache、PHP等组件自行省级。

    对于未购买ECS的用户，可以直接在选择配置时进入镜像市场。已购买ECS者，需先强制停止ECS实例，在配置信息中更换系统盘，进入镜像市场。另外，可以单独进入镜像市场页面购买镜像。

    安全组：

    进入本实例安全组，在配置规则中手动添加安全组。
    对VPC（专有网络）入门者，建议选择入方向开放所有端口并允许所有IP访问：

    对经典网络使用者，只需设置公网出入方向，无需设置内网。

    查看网站：
    访问http://公网IP/，若使用LAMP镜像，即可看到OneinStack初始网页。

    点击探针、phpinfo、phpMyAdmin，可分别查看PHP运行环境、PHP配置信息和MySQL可视化操作界面。

    4.将简历上传到服务器

    安装 FileZilla：

    FileZilla 是支持中文的跨平台FTP软件，支持通过SFTP协议从本地传输文件到虚拟主机（Linux发行版）。

    快速连接:

    • 主机 sftp://公网IP
    • 用户名 root
    • 密码 创建ECS时设置的密码
    • 端口 22

    上传文件：

    • 进入/data/wwwroot/default/
    • 在当前目录新建文件夹，并把所有当前目录的文件移入其中
    • 把简历项目文件拖拽至当前目录

    打开网页：

    • 访问http://公网IP

    打造在线编程环境

    选用Coding WebIDE实现在线编辑，需利用容器服务进行简便的管理集群搭建工作。推荐Docker容器镜像，整合阿里云虚拟化、存储、网络、安全能力——镜像市场中选择Docker运行环境(Ubuntu 16.04 64位)。

    部署Web IDE：

    Coding WeblDE 是 Coding 自主研发的在线集成开发
    环境 (IDE)。用户可以通过 WebIDE 创建项目的工作空间，进行在线开发，调试等操
    作，同时 WebIDE 集成了 Git 代码版本控制，支持绝大部分 Git 托管。

    下载PuTTY传输终端：

    PuTTY可用于Telnet/SSH传输，跨平台，适用于Win32和Unix系统下模拟xterm终端，并且有中文版。

    打开客户端后配置新的会话(session)：

    • 主机名称 公网IP
    • 端口 22
    • 连接类型 SSH

    输入保存的会话后保存，可以在下次使用终端时直接载入会话配置，无须再次输入。
    打开后输入用户名root和ECS实例密码。

    在ECS上安装并运行容器化WebIDE：

    在本地浏览器打开公网IP:端口号即可进入WebIDE。

    使用WebIDE：

    网页上，点击 Show public key，查看IDE公钥。
    进入GitHub，在 Settings > Deploy keys 中添加公钥。
    复制GitHub仓库的SSH地址到WebIDE中，点击Creating。
    加载后点击Open，即可进入IDE。

    添加文件并保存后，菜单栏 版本 > 提交 中填入提交信息，按 版本 > 推送 把文件同步到GitHub仓库。

    基于ECS搭建云上博客

    更换镜像为Windows操作系统（公共镜像Windows Server > 2012 标准版 中文版 64位）。

    1.在本地Windows电脑上连接ECS实例

    Win+R > 输入 mstsc，打开远程桌面连接。
    首次使用时打开显示选项 > 本地资源 > 勾选 剪贴板。
    计算机处输入公网IP。

    2.安装环境

    各安装连接：

    • Visual C++ Redistributable for Visual Studio 2012 Update 4(x64)
    • Visual C++ Redistributable for Visual Studio 2015
    • PHP环境（需先安装Visual C++）
    • MySQL 5.5
    • EMLOG博客程序

    安装IIS：

    • 服务管理器
    • 添加角色和功能
    • 基于角色或基于功能的安装
    • 从服务器池中选择服务器
    • 勾选Web服务器（IIS）
    • 勾选包括管理工具（如适用） > 添加功能
    • 勾选Web服务器（IIS）
    • 保持默认半勾选.NET Framework 4.5 功能（2个已安装，共7个）
    • 全选Web服务器
    • 保持默认勾选，额外勾选CGI，全选管理工具

    打开IE浏览器访问http://127.0.0.1/，可以访问页面，显示Internet Information Services。

    3.安装PHP

    先依次安装两项Visual Studio C++。
    在C:盘目录下创建php7.2目录，解压PHP下载文件后放入其中。

    重命名php.ini-production为php.ini，以记事本打开，取消注释以下文字

    4.配置IIS

    开始菜单点击管理工具，打开Internet Information Services (IIS) 管理器。

    • 起始页 > 处理程序映射
    • 添加模块映射
      • 请求路径 *.php
      • 模块 FastCgiModule
      • 可执行文件（可选） C:\php7.2\php-cgi.exe
      • 名称 FastCGI
    • 是
    • 起始页 > FastCGI 设置
    • 右键C:\php7.2\php-cgi.exe 编辑…
      • 监视对文件所做的更改 C:\php7.2\php.ini
      • 环境变量 ... > 添加
        • Name PHP_FCGI_MAX_REQUESTS
        • Value 1000
    • 起始页 > 默认文档
    • 右侧操作 添加
      • 名称 index.php

    5.测试PHP是否正常运行

    • 起始页 > 网站 > 右键Default Web Site > 浏览
    • 删除两个文件
    • 新建index.php

    7.安装MySQL数据库

    • Typical
    • 勾选Launch
    • Next
    • Detailed Configuration > Next
    • Developer Machine > Next
    • Multifunctional Database > Next
    • C: Installation Path > Next
    • Decision Support (DSS)/OLAP > Next
    • 勾选 Enable TCP/IP Networking，输入3306、Enable Strict Mode > Next
    • Manual Selected Default Character Set / Collation，下拉utf8 > Next
    • 勾选 Install As Windows Service，下拉MySQL、Launch the MySQL Server automatically、Include Bin Directory in Windows PATH > Next
    • 勾选 Modify Security Settings，输入密码
    • Execute > Finish

    8.安装Emlog博客程序

    打开C:\inetpub\wwwroot，删除index.php，把Emlog下载文件解压后放入目录中。

    • 右键wwwroot目录
    • 属性 > 安全 > 编辑 > 添加 > 高级
      • 选择此对象类型 用户、组或内置安全主体
      • 立即查找
      • 选择IUSR
    • 勾选修改项允许

    用管理员模式打开PowerShell

    创建数据库

    登录阿里云ECS控制台，在本实例安全组下配置规则 > 快速创建规则

    • 规则方向 入方向
    • 授权策略 允许
    • 常用端口(TCP) 勾选 HTTP (80)
    • 自定义端口 TCP
    • 优先级 1
    • 授权类型 IPv4地址段访问
    • 授权对象 0.0.0.0/0

    在浏览器访问公网IP/install.php即可访问EMLOG安装程序，按提示输入配置参数，即安装成功。

    拓展：域名解析配置

    国内域名需要备案。

    在阿里云控制台中，域名栏目下点击解析 > 添加记录

    • 记录类型 A
    • 主机记录 为二级域名前缀
    • 解析线路 默认
    • 记录值 公网IP
    • TTL 10分钟

    在远程桌面中，右键Default Web Site > 编辑绑定 > 添加

    • 主机名 主机记录.域名

    即可透过域名访问博客。
